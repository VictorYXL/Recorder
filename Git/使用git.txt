新建git环境：
	git init
	
添加远程仓库：
	git remote add origin git@github.com:VictorYXL/XXX.git
	
从工作区添加到暂存区：
	git add *
	
从暂存区删除：
	git rm XXX --cached
	
从工作区和暂存区删除：
	git rm XXXs
	
查看暂存区内容：
	git ls-files
	
把暂存区的更改提交到版本：
	git commit -m "XXXXXX"
	
查看提交记录:
	git log
	
恢复到之前的提交：
	git reset --hard "commit id"
	
把版本区内容推送到仓库：
	git push origin xxxbranch
	
撤销commit:
	git reset --hard <commit_id>
    git push origin HEAD --force
	
从仓库拉去到工作区:
	git pull
	
切换到某分支:
	git checkout xxx

升级子模块：
	git submodule update --init --recursive 
	
git中文名支持:
	git config --global core.quotepath false
