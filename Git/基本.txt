新建git环境：
	git init
	
添加远程仓库：
	git remote add origin git@github.com:VictorYXL/XXX.git
		
把版本区内容推送到仓库：
	git push origin xxx_branch
	
从仓库拉去到工作区:
	git pull

清理git:
    git gc (--prune=now)

升级子模块：
	git submodule update --init --recursive 

git中文名支持:
	git config --global core.quotepath false
